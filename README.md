# HaxWorks
The only working assworks hack I could find.
<br>
<br>
<br>
Instructions:
1. Install the <a href="./bookmarklet.js">bookmarklet</a> & run it on Classworks.
2. Fetch answer file (automatic fetching is yet to be implemented).
<br>
Notes:
<br>
-Due to the way question autodetection works, you will have to reload the page after every question.
<br>
-Currently, this script is not designed for anything beyond language arts & math. However, it likely can support others (with some help).
<br>
-Lesson skipping has not been added yet.


# Questions
Q: What is an 'answer file'?
<br>
A: 'Answer file' is the name I have given to denote the file that stores information on all the questions in a lesson—including text, answer choices, & of course, the correct answer.
<br>
<br>
Q: Where do I find the answer file?
<br>
A: Open devtools & navigate to <i>Application</i> > <i>Top</i> > <i>assessmentEngine.html</i> > <i>assessmentEngine (assessmentEngine.html)</i> > <i>JavaScript</i> > <i>AssessmentJSONPStorageServlet</i>. If it doesn't appear, reload the page.
